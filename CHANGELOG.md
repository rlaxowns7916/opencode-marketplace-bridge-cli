# Changelog

All notable changes to this project are documented in this file.

The format is based on Keep a Changelog and this project follows Semantic Versioning.

## [0.1.0] - 2026-02-14

### Added
- OpenCode bridge CLI for Claude Code marketplace/plugin sources.
- Install flow with source resolution (`owner/repo`, Git URL, local path).
- Smart cache of referenced directories only.
- Content transformations for path rewrite, `tools`, and `model` normalization.
- Registry-based install/uninstall lifecycle with ownership markers and conflict handling.
- End-to-end and unit test suite for CLI behavior.
